# ProjetoAPI

API REST desenvolvida em ASP.NET Core utilizando Entity Framework e MySQL.

## Tecnologias utilizadas

- ASP.NET Core
- Entity Framework Core
- MySQL
- Pomelo.EntityFrameworkCore.MySql
- Swagger

## Funcionalidades

- CRUD de usuários
- Cadastro de usuário
- Listagem de usuários
- Atualização de usuário
- Remoção de usuário
